## ContGen
Página de creación de posts para CM's (Content Managers). El usuario crea un perfil para una empresa con su respectiva descripción, luego aparece una interfaz  con un 'drag & drop' donde el usuario arrastra las fotos básicas que tiene que tener la publicación final y un recuadro de texto donde el usuario de los prompts para la edición mediante IA de la foto. La IA pensada para esto es Nano Banana de Google. Otra función es que la IA (gemini, openAI, perplexity, …) busque las mejores etiquetas, hashtags, música en base a la publicación y las tendencias en las redes sociales para recomendárselas al usuario y escribirle una descripción para la publicación.

Se puede ver si la API de Instagram/Tiktok permite publicar publicaciones en la cuenta del usuario
