¿Qué es Chef Pro?

TFG | Proyecto Final de Grado

Chef Pro es una aplicación web creada como parte de un Trabajo de Fin de Grado del Grado Superior en Desarrollo de Aplicaciones Web. Su objetivo es conectar a chefs y cocineros autónomos con personas que buscan servicios culinarios personalizados bajo demanda, de forma accesible y flexible.

La plataforma permite reservar servicios, gestionar disponibilidades, coordinar citas y valorar experiencias, ofreciendo funcionalidades tanto para chefs como para comensales. El proyecto ha sido desarrollado aplicando principios de arquitectura web modular, diseño responsive e integración entre frontend, backend, APIs, bases de datos y despliegue.

Aviso importante: Chef Pro no es una empresa real ni ofrece servicios comerciales. Se trata exclusivamente de un proyecto académico con fines educativos y demostrativos.
